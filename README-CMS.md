# Mini CMS for Ybarra Motor

Это отдельная Python-админка для редактирования текста и фото без доступа к стилям и структуре сайта.

Что она делает:

- даёт вход по логину и паролю
- импортирует текущий контент сайта в SQLite
- позволяет редактировать поля страниц через `/admin`
- позволяет загружать новые изображения
- после сохранения сразу пересобирает статические HTML-файлы

Публичный сайт остаётся обычным набором HTML-файлов. Это значит:

- посетители получают быстрый статический сайт
- клиент меняет только содержимое
- CSS, layout и JS не редактируются из админки

## Быстрый старт

Инициализация:

```bash
python3 -m cms.manage init
```

Запуск админки локально:

```bash
python3 -m cms.app
```

Адрес:

```text
http://127.0.0.1:8081/admin
```

Логин по умолчанию:

```text
admin
```

Пароль по умолчанию:

```text
changeme123
```

Сразу после первого запуска смените пароль:

```bash
python3 -m cms.manage set-password YOUR_NEW_PASSWORD
```

## Как встроить на сервер

Нужен любой WSGI-совместимый запуск Python-приложения. Публичный сайт продолжает обслуживаться как раньше из папки `public`, а `cms.wsgi` можно подключить на маршрут `/admin`.

Схема:

- `/` и `/servicios/...` обслуживаются как обычные статические файлы
- `/admin` проксируется в Python WSGI-приложение

Если нужно, следующим шагом можно подготовить конкретный конфиг под `nginx + gunicorn` или под ваш текущий сервер.
